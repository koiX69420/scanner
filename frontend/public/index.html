Getting Started with Mandog Trench Tools Chrome Browser Extention

  1. Install the browser extension - TBA
  2. Navigate to mandog.fun using your browser or click the link within the extension.
  3. If you are not already verified, you will need to do so before the tool will work. This can be done via the official telegram @ManDogMFbot with the /veriy command.
  4. Navigate to the link provided by the Mandog Telegram bot. This will automatically initiate a transaction to pay for 30 days of access.
  5. You can only verify one browser at a time, you can change your device via the official telegram @ManDogMFbot with the /deviceupdate command.
  6. Clicking on the Mandog extension will now show a field to enter a Solana CA (Contract Address). It will also be integrated as a clickable icon on Photon, Bullx and Axiom (Token Page).
  7. You are now ready to begin scanning. Good luck and happy trenching!
